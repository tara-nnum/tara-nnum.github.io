What I Do

💧 Surface Water Modelling

1D/2D hydrodynamic modelling using HEC-RAS and other open-source tools to analyse river flow and flood behaviour.

🗺️ GIS & Spatiotemporal Analysis

Processing, mapping, and analysing spatial datasets using QGIS, ArcGIS, and Python geospatial libraries.

⚙️ Data Pipeline Automation

Developing reproducible Python workflows for data cleaning, automation, and environmental data pipelines.

🌍 Groundwater Modelling

Building conceptual and numerical groundwater models with MODFLOW and related tools for aquifer assessment.

Featured Projects

📘 Representative Year Selection Framework

Developed a data-driven pipeline for representative-year selection using climate model data.

🧩 Image-to-Hexcode Converter

A simple image-processing tool that converts Terrameter readings to hex colour codes for better visual interpretation.

🗞️ Flood Risk Mapping in Tyne Catchment

Generated synthetic rainfall and discharge data of over 10000 years to use as inputs in TELEMAC-2D and simulate a flood inundation map followed by flood risk map of the area.

🗞️ NLP-based Flood News Extractor

Automated extraction of rainfall and flood details from unstructured web articles using Python and NLP techniques.
